RTFTDIF -> Remember These Files To Delete In Future
===================================================
This is a tiny utility appliation to store the liist of temporariily usable files/folders which may not be needed anymore in future. 
If these files/folders gets mixed with other, sometimes it might take more time to find these files. So, whenever u decide certain file/folder is not needed in future, simply
put it in this tiny application. Its just like a notes taking application with simple inbuilt functions to delete those files from the application.
